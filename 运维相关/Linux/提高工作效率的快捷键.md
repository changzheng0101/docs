# 和命令行交互

|      命令       |                 作用                  |
| :-------------: | :-----------------------------------: |
|   `crtl + a`    |              移动到句首               |
|   `crtl + e`    |              移动到句尾               |
|   `crtl + w`    |             删除一个单词              |
|   `crtl + u`    |            清除这一行命令             |
|      上下       |             查看历史命令              |
|      `!+t`      | 在历史命令里面找，第一个以t开头的命令 |
|    `crtl+d`     |               关闭窗口                |
|    `crtl+s`     |     保存当前程序，按下任意键继续      |
|    `crtl+k`     |            删除光标到末尾             |
| `alt+backspace` |             删除一个单词              |
